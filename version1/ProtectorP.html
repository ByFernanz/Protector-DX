Jugar a "El Protector"

  • Requiere habilitar JavaScript.
  • Si no sabes cómo jugar a una aventura, puedes leer esta ayuda primero.
  • Durante el juego, ingresa el comando Ayuda para instrucciones generales; si estás muy atascado en algún punto, el comando Pistas te dará (gradualmente...) indicaciones adicionales para seguir adelante.
  • Los comandos Save o Guardar permiten guardar tu "situación" en el relato y puedes recuperarla con Load o Recuperar, pero no se generarán archivos externos: si cierras el browser, "refrescas" la pantalla o vas a otro enlace y luego vuelves, tendrás que comenzar de nuevo. Si quieres guardar tu partida en archivos externos, necesitas instalar un intérprete de Inform: mira las instrucciones para mayor detalle.
