Instrucciones

Para jugar a "El Protector":

  1. Si no quieres instalar/descargar intérpretes:
    • Juega con Parchment: puedes usar tu navegador para disfrutar de este relato interactivo. Requiere habilitar JavaScript.
    • Juega con Java: ejecuta el programa zmpp.jar adjunto y luego abre el archivo Protector.z5 adjunto. Necesitas tener instalado un Java Runtime.
  2. Si quieres jugar en tu dispositivo (sin navegador) puedes:
    • Descargar e instalar un intérprete de Inform (compatible con Windows, LINUX, Android o qué sé yo) para leer el archivo Protector.z5 del juego.
    • Una vez instalado, ejecuta con el intérprete el archivo Protector.z5 adjunto.
    • Opcionalmente, puedes usar el archivo Papyrus.ttf adjunto, para ver los textos del juego con la fuente de la imagen de título de esta página.
    • Si usas el intérprete Gargoyle, puedes usar si lo deseas el archivo de configuración Protector.ini adjunto.
  3. Si no sabes cómo jugar a una aventura, puedes leer esta ayuda primero.
  4. Durante el juego, ingresa el comando Ayuda para instrucciones generales; si estás muy atascado en algún punto, el comando Pistas te dará (gradualmente...) indicaciones adicionales para seguir adelante.
